# Poem Generator

A simple command-line poem generator that creates mood-based poems at different lengths.

## Features

- Choose from multiple moods: sad, romantic, feel good, or mysterious.
- Select poem length: short, medium, long, or very long.
- Generates varied lines with mood-specific imagery.

## Requirements

- Python 3.10+ (works with any modern Python 3 release).

## Usage

Run the script from the repository root:

```bash
python3 "Poem Generator"
```

Follow the prompts to pick a length and mood. The poem will be printed to the terminal.

## Project structure

- `Poem Generator`: The main executable script that handles prompts and poem generation.
